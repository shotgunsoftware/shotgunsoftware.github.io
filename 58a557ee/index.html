Community Shared Integrations

Here are projects that people in the Toolkit community have been gracious enough to share. They arent written by us at Flow Production Tracking and we cant vouch for them, but well definitely answer any questions we can about them. If you have a project you want added to the list, please visit our support site for help.

Project Description: Getting Flow Production Tracking data into nuke text nodes can be a bit of a pain. This node can be used to pipe that info into slates, burn-ins, or anywhere else.
