Ж░юВџћ

Ж░ёьў╣ ShotGridВЮў ВЋАВЁў вЕћвЅ┤ВЌљ ВѓгВџЕ Ж░ђвіЦьЋю ьѕ┤ьѓи ВЮЉВџЕьћёвАюЖиИвъе вфЕвАЮВЮ┤ ьЉюВІювљўВДђ ВЋіЖ▒░вѓў, ShotGrid вЇ░ВіцьЂгьє▒ВЮ┤ ВІцьќЅ ВцЉВЮИвЇ░вЈё вАюВ╗г ьїїВЮ╝ вДЂьЂгвЦ╝ ВѓгВџЕьЋўвЕ┤ ВўцвЦўЖ░ђ в░юВЃЮьЋа Вѕў ВъѕВіхвІѕвІц. ВЮ┤ вгИВёювіћ ChromeВЌљВёю ShotGrid вЇ░ВіцьЂгьє▒ВЮў вИївЮ╝Вџ░Вађ ьєхьЋЕ вгИВаювЦ╝ ьЋ┤Ж▓░ьЋўЖ│а, ВЏљьЋўвіћ ЖИ░віЦвЊцВЮ┤ ВаювїђвАю ВъЉвЈЎьЋўвЈёвАЮ ВДђВЏљьЋўвіћ ВЋѕвѓ┤ВёюВъЁвІѕвІц. вгИВаю ВДёвІе

ShotGrid вЇ░ВіцьЂгьє▒ВЮ┤ ВІцьќЅ ВцЉВъЁвІѕЖ╣ї?

Вўѕ, ВЋїЖ│а ВъѕВіхвІѕвІц. ВЮ┤в»И ьЎЋВЮИьЋўВЁеВЮё Ж▓ЃВЮ┤вЮ╝ ВЃЮЖ░ЂьќѕВДђвДї вІцВІю ьЋюв▓ѕ ьЎЋВЮИьќѕВіхвІѕвІц. :)

ShotGrid вЇ░ВіцьЂгьє▒ВЮё ВІюВъЉьЋю ьЏёВЌљ ChromeВЮё вІцВІю ВІюВъЉьќѕВіхвІѕЖ╣ї?

ShotGrid вЇ░ВіцьЂгьє▒ВЮ┤ ВЮИВдЮВёювЦ╝ вЊ▒вАЮ(ShotGrid вЇ░ВіцьЂгьє▒ВЮё В▓ўВЮї ВІцьќЅьЋа вЋїВЌљвДї вЊ▒вАЮьЋўвЕ┤ вљўЖ│а, ВЮ┤ьЏёвХђьё░віћ вгИВаю ВЌєВЮї)ьЋўЖИ░ ВаёВЌљ ChromeВЮё ВІцьќЅьќѕвІцвЕ┤ ChromeВЮ┤ Вўцвъўвљю ВЮИВдЮВёю Вѓгв│ИВЮё ВѓгВџЕьЋўвЕ┤Вёю ShotGrid вЇ░ВіцьЂгьє▒ ВЌ░Ж▓░ВЮё Ж▒░вХђьЋЕвІѕвІц. вфевЊа ьЃГВЮё вІФвіћвІцЖ│а ьЋ┤Вёю ChromeВЮ┤ вІФьъѕвіћ Ж▓ЃВЮђ ВЋёвІѕв»ђвАю ВБ╝Вєї ьЉюВІюВцёВЌљ chrome://restartвЦ╝ ВъЁваЦьЋўЖ│а Enter ьѓцвЦ╝ вѕёвЦ┤віћ Ж▓ЃВЮ┤ ВбІВіхвІѕвІц. ВЮ┤ваЄЖ▓ї ьЋўвЕ┤ вфевЊа Chrome Ж┤ђвае ьћёвАюВёИВіцЖ░ђ ВбЁвБївљўЖ│а, ChromeВЮ┤ вІцВІю ВІюВъЉвљЕвІѕвІц.

в░ЕьЎћв▓й ВєїьћёьіИВЏеВќ┤вЦ╝ ВѓгВџЕ ВцЉВъЁвІѕЖ╣ї?

в░ЕьЎћв▓й ВєїьћёьіИВЏеВќ┤Ж░ђ localhost вўљвіћ ьЈгьіИ 9000 ВаЉВєЇВЮё В░евІе ВцЉВЮ┤ВДђ ВЋіВЮђВДђ ьЎЋВЮИьЋўВІГВІюВўц.

ChromeВЮ┤ ВЮИВдЮВёювЦ╝ Ж▒░вХђьЋЕвІѕЖ╣ї?

https://localhost:9000Вю╝вАю ВЮ┤вЈЎьЋўВЌг ChromeВЮ┤ ВЮИВдЮВёювЦ╝ ВѕўвЮйьЋўвіћВДђ ьЎЋВЮИьЋа Вѕў ВъѕВіхвІѕвІц. ВЮ┤ ВБ╝Вєївіћ ShotGrid ВЏ╣ ВѓгВЮ┤ьіИЖ░ђ вАюВ╗г ьїїВЮ╝ вДЂьЂгвЦ╝ ВѕўьќЅьЋўЖ│а ьѕ┤ьѓи ВЮЉВџЕьћёвАюЖиИвъеВЮё ВІцьќЅьЋўЖИ░ ВюёьЋ┤ ВЋАВёИВіцьЋўвіћ ВБ╝ВєїВъЁвІѕвІц. в│┤ьєх вІцВЮї вЕћВІюВДђЖ░ђ вѓўьЃђвѓўВЋ╝ ьЋЕвІѕвІц.

ВЮ┤ВЎђ вІгвдг вІцВЮї вЕћВІюВДђ ВцЉ ьЋўвѓўЖ░ђ ьЉюВІювљювІцвЕ┤ ВЮИВдЮВёю вЊ▒вАЮ ьћёвАюВёИВіцВЌљ вгИВаюЖ░ђ в░юВЃЮьќѕвІцвіћ ВЮўв»ИВъЁвІѕвІц.

ВЮИВдЮВёю вгИВаювЦ╝ в╣авЦ┤Ж▓ї ьЋ┤Ж▓░ьЋўвіћ в░Ев▓Ћ

ВЮ┤ вгИВаювЦ╝ в░ЕВДђьЋўвіћ Ж░ђВъЦ ВЅгВџ┤ в░Ев▓ЋВЮђ Ж│аЖИЅ(ADVANCED), localhostвАю ВДёьќЅ(Proceed to localhost)(ВЋѕВаёьЋўВДђ ВЋіВЮї)ВЮё ьЂ┤вдГьЋўвіћ Ж▓ЃВъЁвІѕвІц. ВЮ┤ваЄЖ▓ї ьЋўвЕ┤ ChromeВЌљВёю ВЌгвЪгвХёВЮ┤ ВЮИВдЮВёювЦ╝ ВѕўвЮйьЋўвацЖ│а ьЋювІцвіћ Ж▓ЃВЮё ВЮИВІЮьЋўЖ│а, ShotGrid ВЏ╣ ВѓгВЮ┤ьіИЖ░ђ ShotGrid вЇ░ВіцьЂгьє▒Ж│╝ ьєхВІаьЋўвЈёвАЮ ьЌѕВџЕьЋЕвІѕвІц.

В░ИЖ│а : ВЏ╣ вИївЮ╝Вџ░ВађВЎђ ShotGrid вЇ░ВіцьЂгьє▒ Ж░ёВЌљ ВЌ░Ж▓░ВЮ┤ ВЮ┤вБеВќ┤ВДђвЈёвАЮ ьЋа Вѕў Въѕвіћ в░Ев▓ЋВЮђ ВЮ┤Ж▓Ѓв┐љВъЁвІѕвІц. ВІавб░ ВёцВаЋВЮђ вЉљ ВЮЉВџЕьћёвАюЖиИвъе Ж░ё ьіИвъўьћй ьЮљвдёВЮё Ж░ђвіЦьЋўвЈёвАЮ ьЋа в┐љ, ВЮИьё░вёи ВЃЂВЮў вІцвЦИ вфевЊа Вёюв▓ёВЌљ вїђьЋю ВІавб░вЦ╝ ВЮўв»ИьЋўвіћ Ж▓ЃВЮђ ВЋёвІЎвІѕвІц. ВЋёВЅйЖ▓ївЈё ВЮ┤ ьћёвАюВёИВіцвіћ вгИВаюЖ░ђ Въѕвіћ вфевЊа В╗┤ьЊеьё░вДѕвІц в░ўв│хьЋ┤ВЋ╝ ьЋЕвІѕвІц. ВЮ┤ваЄЖ▓ї ьЋ┤вЈё вгИВаюЖ░ђ ьЋ┤Ж▓░вљўВДђ ВЋіЖ▒░вѓў вфевЊа ВѓгВџЕВъљвЊцВЌљЖ▓ї ВЋївацВБ╝ЖИ░ВЌљвіћ вёѕвг┤ в│хВъАьЋю в░Ев▓ЋВЮ┤вЮ╝Ж│а ВЃЮЖ░Ђвљўвіћ Ж▓йВџ░ вІцВЮї вІеЖ│ёвЦ╝ Вѓ┤ьј┤в│┤ВІюЖИ░ в░ћвъЇвІѕвІц.

В░ИЖ│а : ВЮ┤ вІеЖ│ё ьЋ┤Ж▓░ в░Ев▓ЋВЮђ ERR_SSL_SERVER_CERT_BAD_FORMAT ВўцвЦўВЌљвіћ ВаЂВџЕвљўВДђ ВЋіВю╝вЕ░, ВЋёвъў ВёцвфЁВЌљ вћ░вЮ╝ ВЮИВдЮВёювЦ╝ вІцВІю ВЃЮВё▒ьЋ┤ВЋ╝ ьЋЕвІѕвІц.

вфевЊа ьћївъФьЈ╝ВЌљВёю NET::ERR_CERT_COMMON_NAME_INVALID в░Ј ERR_SSL_SERVER_CERT_BAD_FORMAT ьЋ┤Ж▓░

ChromeВЮђ ВъљВ▓┤ ВёювфЁ ВЮИВдЮВёюВЌљ Ж┤ђьЋю в│┤ВЋѕВЮё ВаЋЖИ░ВаЂВю╝вАю ВЌЁЖиИваѕВЮ┤вЊюьЋўвЕ░, ВЮ┤ ВЌЁвЇ░ВЮ┤ьіИвАю ВЮИьЋ┤ ВбЁВбЁ вИївЮ╝Вџ░Вађ ьєхьЋЕВЮ┤ вЂіВќ┤ВДЉвІѕвІц. вХѕьќЅьъѕвЈё ВЮ┤вЪ░ ВбЁвЦўВЮў вгИВаювіћ ВЮИВдЮВёюВЮў certificate_pathвЦ╝ вІцВІю ВЃЮВё▒ьЋўвіћ в░Ев▓ЋВю╝вАювДї ьЋ┤Ж▓░ьЋа Вѕў ВъѕВіхвІѕвІц.

ВЮИВдЮВёювЦ╝ вІцВІю ВЃЮВё▒ьЋўвацвЕ┤ ShotGrid вЇ░ВіцьЂгьє▒ВЮў ВѓгВџЕВъљ вЕћвЅ┤ВЌљ Въѕвіћ Ж│аЖИЅ(Advanced) Вё╣ВЁўВЌљВёю ВЮИВдЮВёю ВъгВЃЮВё▒(Regenerate Certificates) ВўхВЁўВЮё ВёаьЃЮьЋ┤ВЋ╝ ьЋЕвІѕвІц. (ВЮ┤ ВўхВЁўВЮ┤ в│┤ВЮ┤ВДђ ВЋіВю╝вЕ┤ tk-desktop ВЌћВДёВЮё ВЌЁвЇ░ВЮ┤ьіИьЋўВЌг ВъаЖИѕ ьЋ┤ВаюьЋЕвІѕвІц.)

ВЮИВдЮВёю ВъгВЃЮВё▒ВЮё ьЎЋВЮИьЋўЖ│а вѓўвЕ┤ ВЮИВдЮВёювЦ╝ В▓ўВЮї ВЃЮВё▒ьќѕВЮё вЋїВ▓ўвЪ╝ ВЮ╝ваеВЮў вїђьЎћВЃЂВъљЖ░ђ вѓўьЃђвѓЕвІѕвІц. Windows в░Ј macOSВЌљВёювіћ Windows ВЮИВдЮВёю ВађВъЦВєї вўљвіћ macOS ьѓцВ▓┤ВЮИВЮё ВЌЁвЇ░ВЮ┤ьіИьЋўвЮ╝віћ вЕћВІюВДђЖ░ђ вЉљ в▓ѕ вѓўьЃђвѓЕвІѕвІц. ьЋю в▓ѕВЮђ ВЮ┤Ваё ВЮИВдЮВёювЦ╝ ВаюЖ▒░ьЋўвЮ╝віћ вЕћВІюВДђЖ░ђ вѓўьЃђвѓўЖ│а, ВЃѕ ВЮИВдЮВёювЦ╝ вЊ▒вАЮьЋўвЮ╝віћ вЕћВІюВДђЖ░ђ ьЋю в▓ѕ вЇћ вѓўьЃђвѓЕвІѕвІц. LinuxВЌљВёювіћ вЊ▒вАЮВЮ┤ ВъљвЈЎВю╝вАю ВЮ┤вБеВќ┤ВДЉвІѕвІц. вЊ▒вАЮВЮ┤ вЂЮвѓўвЕ┤ ShotGrid вЇ░ВіцьЂгьє▒ВЮё вІцВІю ВІюВъЉьЋЕвІѕвІц.

ShotGrid вЇ░ВіцьЂгьє▒ВЮ┤ вІцВІю ВІюВъЉвљўВќ┤ ВІцьќЅвљўвЕ┤ ВБ╝Вєї ьЉюВІюВцёВЌљ chrome://restartвЦ╝ ВъЁваЦьЋ┤ ChromeВЮё вІцВІю ВІюВъЉьЋеВю╝вАюВЇе ChromeВЮ┤ ВЎёВаёьъѕ ВбЁвБївљўЖ│а ВЮИВдЮВёю В║љВІюЖ░ђ ВДђВЏїВДђвЈёвАЮ ьЋўвіћ Ж▓ЃВЮ┤ ВбІВіхвІѕвІц.

В╗┤ьЊеьё░Ж░ђ ВЮИьё░вёиВЌљ ВЌ░Ж▓░вљўВќ┤ ВъѕВДђ ВЋіВЋё ВЌЁвЇ░ВЮ┤ьіИвЦ╝ вІцВџ┤вАювЊюьЋа Вѕў ВЌєвІцвЕ┤ support@shotgunsoftware.comВЌљ вгИВЮўьЋўВІюЖИ░ в░ћвъЇвІѕвІц.

WindowsВЌљВёю NET::ERR_CERT_AUTHORITY_INVALID ьЋ┤Ж▓░

Ж▓йВџ░ВЌљ вћ░вЮ╝ WindowsЖ░ђ ВЮИВдЮВёювЦ╝ Вё▒Ж│хВаЂВю╝вАю Ж░ђВаИВЎћвІцЖ│а в│┤Ж│аьЋўвіћвЇ░ ВЮИВдЮВёюЖ░ђ ьЋёВџћьЋю ВЮЉВџЕьћёвАюЖиИвъеВЌљ ВЮ┤вЦ╝ ВѓгВџЕьЋа Вѕў ВЌєВЮё вЋїЖ░ђ ВъѕВіхвІѕвІц. ВЮ┤віћ WindowsВЌљВёю ВЮИВдЮВёю вїђьЎћВЃЂВъљвАю ВЮ┤вЈЎьЋўВЌг ьЎЋВЮИьЋа Вѕў ВъѕВіхвІѕвІц. вїђьЎћВЃЂВъљВЌљ ВЋАВёИВіцьЋўвацвЕ┤ Windows ьѓцвЦ╝ вѕёвЦ┤Ж│а ВЮИьё░вёи ВўхВЁўВЮё ВъЁваЦьЋЕвІѕвІц. ВЮИьё░вёи ВєЇВё▒ вїђьЎћВЃЂВъљВЌљВёю вѓ┤ВџЕ ьЃГВю╝вАю ВЮ┤вЈЎьЋю вІцВЮї ВЮИВдЮВёю в▓ёьі╝ВЮё ьЂ┤вдГьЋЕвІѕвІц. вДѕВДђвДЅВю╝вАю ВІавб░ьЋа Вѕў Въѕвіћ вБеьіИ ВЮИВдЮ ЖИ░Ж┤ђВЮё ьЂ┤вдГьЋўЖ│а localhostвЦ╝ В░ЙВіхвІѕвІц.

ьЋ┤вІ╣ ьЋГвфЕВЮ┤ ВЌєВю╝вЕ┤ Windows вЈёвЕћВЮИ вўљвіћ вАюВ╗г В╗┤ьЊеьё░ВЌљ ЖиИвБ╣ ВаЋВ▒Ё вгИВаюЖ░ђ ВъѕвІцвіћ ВЮўв»ИВЮ╝ Вѕў ВъѕВіхвІѕвІц. ВЮ┤ вгИВаюЖ░ђ ВъѕВю╝вЕ┤ ВДђВЏљ ьїђВЌљ вгИВЮўьЋўвіћ Ж▓ЃВЮ┤ ВбІВіхвІѕвІц.

ВЮ┤ Ж▓йВџ░, ВА░ВДЂВЮў Windows В╗┤ьЊеьё░ Ж┤ђвдгВъљВЌљЖ▓ї ВЌ░вЮйьЋўВЌг WindowsвАю ьЋўВЌгЖИѕ ВъљВ▓┤ ВёювфЁ ВЮИВдЮВёювЦ╝ ВІавб░ьЋўВДђ ВЋівЈёвАЮ ВДђВІюьЋўвіћ ЖиИвБ╣ ВаЋВ▒ЁВЮ┤ ВёцВаЋвљўВќ┤ ВъѕВДђ ВЋіВЮђВДђ вгИВЮўьЋ┤ в│┤ВІГВІюВўц.

ЖиИвБ╣ ВаЋВ▒Ё вгИВаю ВА░Вѓг
