환영합니다!

이 사이트에서는 개발자가 Flow Production Tracking 웹 앱, 아티스트 파이프라인 빌드를 위한 툴킷 플랫폼 및 디지털 리뷰 도구의 RV 제품군과 상호 작용할 수 있는 모든 방법을 소개합니다. 안내서 섹션에는 튜토리얼, 가이드 및 기타 교육 리소스가 포함되어 있고 참조 섹션에는 API 및 구성 파일에 대한 자세한 기술 설명서가 포함되어 있습니다.

이 사이트 사용 대상은 누구입니까?

이 사이트의 자료는 소프트웨어 엔지니어, 파이프라인 TD 또는 파이프라인을 설정하고 Flow Production Tracking 및 RV와 관련된 도구를 개발하는 사용자를 위해 작성되었습니다.

Flow Production Tracking 사용에 대한 일반적인 도움말은 Flow Production Tracking 도움말 포털을 참조하십시오.

ShotGrid를 사용하여 파이프라인을 개발하려면 어떻게 해야 합니까?

ShotGrid는 스튜디오의 Flow Production Tracking 경험을 커스터마이즈하기 위한 강력한 도구 세트를 제공합니다. 다음은 그중 일부입니다.

커스텀 코드를 통해 Flow Production Tracking와 상호 작용하는 방법을 알고 싶으십니까?

  • Python API는 Flow Production Tracking 데이터베이스에서 들어오고 나가는 데이터에 대한 원시 액세스를 지원하는 CRUD 방식을 제공합니다.
  • REST API는 Python API와 기능이 동일하지만 Python 이외의 프로그래밍 언어로 Flow Production Tracking에 대한 완전한 데이터 액세스를 제공합니다.
  • Toolkit Core API는 게시 자동화, 파일 위치 처리, 교차 소프트웨어 기능 도구 빌드와 같은 Flow Production Tracking 파이프라인 통합에 대한 인터페이스를 제공합니다.

Flow Production Tracking 웹 앱과 관련된 프로세스를 자동화하는 방법에 대해 알고 싶으십니까?

ShotGrid는 Flow Production Tracking 사이트에 트리거 또는 커스텀 메뉴 항목을 추가하기 위한 몇 가지 기술을 제공합니다.

  • 액션 메뉴 항목(AMI)을 사용하여 엔티티의 컨텍스트 메뉴에 항목을 추가하면 Flow Production Tracking 사용자가 메뉴 항목을 통해 커스텀 로직을 실행할 수 있습니다.
  • 웹후크를 사용하면 Flow Production Tracking 사이트에서 발생하는 이벤트에 대한 커스텀 반응을 생성할 수 있습니다. 선택한 이벤트가 발생하면 사용자가 정의한 엔드포인트에 해당 이벤트에 대한 정보 페이로드가 있는 HTTPS 요청이 발생합니다. 이 페이로드를 사용하여 원하는 방식으로 반응하는 커스텀 코드를 작성할 수 있습니다.
  • 이벤트 데몬 프레임워크는 사이트의 이벤트에 대한 반응을 생성하는 또 다른 솔루션입니다. 들어오는 인터넷 연결을 특정 스튜디오로 제한하려면 이벤트 데몬 프레임워크를 선택합니다.

아티스트 파이프라인으로 Flow Production Tracking를 확장하는 방법에 대해 알고 싶으십니까?

Flow Production Tracking 툴킷은 Flow Production Tracking 환경을 프로덕션 워크플로우로 가져오고 게시 파이프라인의 토대를 제공하는 아티스트 도구 내에서 플러그인 통합을 빌드하는 플랫폼입니다.

다양한 방법으로 툴킷 설정을 커스터마이즈할 수 있습니다.

  • 툴킷 파이프라인 구성은 툴킷 플러그인 앱 내의 다양한 옵션을 제어할 수 있는 YAML 파일 세트입니다.
  • 후크를 사용하면 주 코드 베이스를 포크(fork)할 필요 없이 툴킷 워크플로우의 특정 지점에 커스텀 Python 로직을 주입할 수 있습니다.
  • Toolkit Core API는 파이프라인 TD가 공통 컨텐츠 생성 소프트웨어에서 실행할 구성 가능한 도구를 빠르게 빌드할 수 있는 방식을 제공합니다.
  • 또한 다양한 앱 및 프레임워크 API를 통해 아티스트 작업 게시, UI 위젯 및 다양한 유틸리티 처리 등 특정 툴킷 기능을 처리하는 코드를 작성할 수 있습니다.

스튜디오의 미디어 리뷰 환경을 커스터마이즈하는 방법에 대해 알고 싶으십니까?

RV는 처음부터 확장 가능한 플랫폼으로 빌드되었습니다. 주로 Mu, Python 및 Javascript에서 제공되는 API를 사용하여 빌드된 인터페이스를 통해 자동화를 추가할 수 있는 유연한 시스템을 제공합니다. 다음은 플러그인을 통해 제공되는 커스텀 동작의 몇 가지 예입니다.

  • 색상 파이프라인 및 뷰 관리 자동화
  • RV 내에서 기존 스튜디오 라이브러리를 활용하는 아티스트 대상 도구 제공

다른 사용자들이 Flow Production Tracking를 실제로 사용하는 방법에 대해 알고 싶으십니까?

커뮤니티 포럼은 Flow Production Tracking 직원 및 사용자, 관리자, 매일 Flow Production Tracking로 작업하는 개발자들이 활발하게 교류하는 홈그라운드와 같은 공간입니다.
